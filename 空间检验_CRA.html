    CRA方法

    CRA方法是基于目标识别和刚体变换的方法，诊断要素目标（例如雨带）的位置偏差、形态偏差、强度偏差的一种空间检验方法。CRA的实现步骤包括如下部分：

    1. 根据MODE算法模块中的目标识别、目标匹配和目标合并方法，获取具有匹对关系的观测和预报目标
    2. 根据刚体变换方法（rigider）对每个匹配上的预报目标进行平移和旋转，根据平移和旋转的场计算预报误差的不同分量。
    craer(look_merge, stages = True,translate = True,rotate = True)
    参数 说明
    look_merge MODE算法完成目标识别匹配和合并后的输出结果
    stages 布尔值，该参数为True时，采用方案1同时计算最优的平移和旋转，否作采用方案2
    translate stages = False时有效，当translate为True时代表会计算最优平移，否则不计算
    rotate stages = False时有效，当rotate为True时代表会计算最优旋转，否则不计算

    ————————————————————————————————————————————————————————————————————————————————————————————————————————————

    craer 返回结果是一个两层嵌套的字典，自动的第一级关键词是每个成功匹配的目标的编号，第二级关键词对应该目标的各类检验结果，内容说明如下：
    二级关键词 说明 单位
    grd_ob 从输入数据种继承的二维矩阵形式的网格观测场
    grd_fo 从输入数据种继承的二维矩阵形式的网格预报场
    grd_fo_shift 平移后的预报场，网格数据
    grd_fo_shift_rotate 平移+旋转后的预报场，网格数据
    delta_lon 质心经度偏差（预报-观测） °
    delta_lat 质心纬度偏差（预报-观测） °
    delta_angle 目标倾角偏差（预报-观测） °
    ob_mean 观测场均值 和要素原始单位一致
    fo_mean 原始预报场均值 和要素原始单位一致
    MSE_total 总误差=原始预报和观测之间误差场的均方 要素原始单位的平方，例如当grd_ob是降水场时，单位为mm2
    MSE_shift_left 平移场残差=平移后的预报与原始预报之间误差场的均方 要素原始单位的平方
    MSE_shift 平移误差=MSE_total -MSE_shift_left 要素原始单位的平方
    MSE_shift_rotate_left 平移旋转场残差=平移旋转后的预报与观测之间误差场的均方 要素原始单位的平方
    MSE_rotate 旋转误差= MSE_shift_left - MSE_shift_rotate_left 要素原始单位的平方
    MSE_volume 强度误差 = （预报均值-观测均值）2 要素原始单位的平方
    MSE_pattern 形态误差 = （MSE_shift_rotate_left-MSE_volume） 要素原始单位的平方

    在CRA算法中采用最小的能够覆盖grd_ob、grd_fo、grd_fo_shift、grd_fo_shift_rotate非0值的矩形区域内的格点数作为各项平均值以及均方差统计的样本数。
    如果将预报(fo)、平移后的预报(fo_shift)、平移旋转后的预报(fo_shift_rotate)以及观测(ob)看做是相空间中的一个点，下面通过一张示意图大致说明CRA输出的不同的均方差的含义。

    上面的示意图中MSE_shift 并不等于矢量fo->fo_shift的长度的平方，MSE_rotate 也并不等于矢量fo_shift->fo_shift_rotate的长度的平方，因此上面的示意图是不严谨的，它只是大致示意。

    operation(grd_ob,grd_fo,smooth,threshold,minsize,compare = “>=”, stages = True,translate = True,rotate = True)
    参数 说明
    grd_ob 二维矩阵形式的网格观测场
    grd_fo 二维矩阵形式的网格预报场
    smooth 平滑系数,采用圆盘形的卷积内核对网格数据进行平滑，smooth相当圆盘的半径 可以为单个整数或列表形式，当其为单个整数时，观测和预报采用相同的参数，否则相应采用不同的参数，如smooth = [5, 4]。平滑系数为0时不做平滑
    threshold 雨量阈值，平滑后低于阈值的部分会被置0 可以为单个实数或列表形式，当其为单个实数时，观测和预报采用相同的参数，否则相应采用不同的参数，，如threshold = [9,10]
    minsize 目标的最小面积,面积小于该阈值的目标将会被删除 可以为单个整数或列表形式，当其为单个整数时，观测和预报采用相同的参数，否则相应采用不同的参数，
    如minsize = [5, 4]，此列表中的数值表示的是格点数。
    compare 比较方法，可选项包括”>=”,”>”,”<=”,”<” 当关注目标是要素场大于阈值时,采用默认参数，如识别暴雨目标时选择参数”>=”,若关注对象是要素值小于阈值时，则可选取”<=”，如根据能见度场识别大雾目标
    stages 布尔值，该参数为True时，采用方案1同时计算最优的平移和旋转，否作采用方案2
    translate stages = False时有效，当translate为True时代表会计算最优平移，否则不计算
    rotate stages = False时有效，当rotate为True时代表会计算最优旋转，否则不计算

    一级关键词 说明 单位
    merge look_merge,MODE算法完成目标识别匹配和合并后的输出结果 字典
    cra look_cra, craer 函数的返回结果 字典
