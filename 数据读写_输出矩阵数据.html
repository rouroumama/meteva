输出矩阵数据 
    复制
import meteva.base as meb
import numpy as np

    将数组输出成excel

    write_array_to_excel(array,save_path,name_list_dict,columns = None,index = None)
    参数 说明
    array numpy 矩阵数据，暂时只支持1、2、3维的矩阵
    save_path 数据保存路径，需要以.xlsx作为后缀
    name_list_dict 以字典形式存储的每个维度的描述信息，其中字典的key为维度的名称，字典的value 为维度的坐标值的列表，列表中的元素只支持字符型 和 时间类型
    columns 输出结果中excel表的列名称
    index 输出结果中excel表的行名称
    return 无返回值

    调用示例

    复制#将一维数据输出输出到excel的示例
data1 = np.random.rand(10)  
name_list_dict1 = {"时效(单位：小时)":[3,6,9,12,15,18,21,24,27,30]}
save_path = r"H:\test_data\output\meb\data1.xlsx"
meb.write_array_to_excel(data1,save_path,name_list_dict1)

    复制数据已以excel表格形式保存至H:\test_data\output\meb\data1.xlsx


    复制#将二维数据输出输出到excel的示例

    "时效(单位：小时)":[3,6,9,12,15,18,21,24,27,30]}
